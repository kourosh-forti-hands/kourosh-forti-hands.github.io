## Tech Stack

This portfolio is built with modern web technologies:

- **[Astro](https://astro.build/)** - Static site generator for fast, content-focused websites
- **[Tailwind CSS v4](https://tailwindcss.com/)** - Utility-first CSS framework
- **[TypeScript](https://www.typescriptlang.org/)** - Type-safe configuration
- **[Tabler Icons](https://tabler-icons.io/)** - Modern icon library

## Project Structure

```
├── public/              # Static assets
├── src/
│   ├── components/      # Astro components (Hero, About, Projects, etc.)
│   ├── pages/           # Page routes
│   ├── styles/          # Global styles
│   └── config.ts        # Site configuration (content lives here)
├── astro.config.mjs     # Astro configuration
├── tailwind.config.js   # Tailwind configuration
└── package.json
```

## Development

### Prerequisites

- Node.js 18+
- npm or yarn

### Getting Started

```bash
# Clone the repository
git clone https://github.com/kourosh-forti-hands/kourosh-forti-hands.github.io.git
cd kourosh-forti-hands.github.io

# Install dependencies
npm install

# Start development server
npm run dev
```

### Available Commands

| Command           | Description                              |
|-------------------|------------------------------------------|
| `npm run dev`     | Start local development server           |
| `npm run build`   | Build production site to `./dist/`       |
| `npm run preview` | Preview production build locally         |

## Customization

All content is managed through a single configuration file at `src/config.ts`. This includes:

- **Personal Information** - Name, title, description
- **Social Links** - Email, LinkedIn, GitHub
- **About Me** - Professional summary
- **Skills** - Technical competencies
- **Projects** - Featured work and achievements
- **Experience** - Work history
- **Education** - Academic background
- **Certifications** - Professional certifications
- **Courses** - Continuing education
- **Awards** - Recognition and achievements
